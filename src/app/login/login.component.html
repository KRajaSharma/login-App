All About Using Bootstrap

Bootstrap is a free and open-source front-end web framework. It contains HTML and CSS-based design templates for typography, forms, buttons, navigation and other interface components, as well as optional JavaScript extensions. Unlike many earlier web frameworks, it concerns itself with front-end development only. Bootstrap is the second-most-starred project on GitHub, with more than 131,000 stars.
